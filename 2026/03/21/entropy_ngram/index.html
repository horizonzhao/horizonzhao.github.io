杂货铺1：熵，KL散度，N-gram语言模型

Posted on 2026-03-21 In 技术分享
熵、交叉熵、KL散度在机器学习中的应用，再聊聊自然语言处理的N-gram。

熵

  • 热力学中，熵是分子混乱程度的度量，香农用信息熵表示随机变量的不确定度，一个离散型的随机变量XXX的熵为：
H(X)=−∑x∈Xp(x)logp(x) H(X) = -\sum_{x \in X} p(x) log p(x) H(X)=−x∈X∑​p(x)logp(x)

本质就是数学期望E[−log(P(X))]E[-log(P(X))]E[−log(P(X))]

两个理解角度

1. 不确定性

熵越高，表示系统的不确定性越大，或者说系统的「混乱度」越高。
e.g. 一枚均匀的硬币的熵比一枚不均匀的硬币的熵要高，因为前者更难预测。

2. 信息量

熵也可以理解为描述该随机变量所需的最小平均信息量。

  • 如果一个事件发生的概率很低，一旦发生，它携带的信息量就很大：−log⁡P(x)-\log P(x)−logP(x)
  • 如果概率很高，携带的信息量就小。

熵的性质

  1. 非负性：H(X)≥0H(X) \ge 0H(X)≥0。
  2. 最大值：当所有事件概率相等时（均匀分布），熵达到最大值。
  3. 最小值：当某个事件概率为1，其余为0时（确定性事件），熵为0。

KL散度

DKL(P∥Q)=−∑i=1nP(xi)log⁡Q(xi)−(−∑i=1nP(xi)log⁡P(xi)) D_{KL}(P \parallel Q) = -\sum_{i=1}^{n} P(x_i) \log Q(x_i) - \left( -\sum_{i=1}^{n} P(x_i) \log P(x_i) \right) DKL​(P∥Q)=−i=1∑n​P(xi​)logQ(xi​)−(−i=1∑n​P(xi​)logP(xi​))

进一步简化为：

DKL(P∥Q)=H(P,Q)−H(P) D_{KL}(P \parallel Q) = H(P, Q) - H(P) DKL​(P∥Q)=H(P,Q)−H(P)
  • 衡量两个分布 P 和 Q 之间差异的非对称性度量，使用基于分布 Q 的编码来编码来自分布 P 的数据时，额外需要的比特数。DKL(P∣∣Q)D_{KL}(P||Q)DKL​(P∣∣Q) 越小，说明 QQQ 越接近 PPP。
  • H(P,Q)H(P,Q)H(P,Q)称为交叉熵

KL散度的性质

DKL(P∥Q)=H(P,Q)−H(P) D_{KL}(P \parallel Q) = H(P, Q) - H(P) DKL​(P∥Q)=H(P,Q)−H(P)

  1. 非对称性：DKL(P∣∣Q)≠DKL(Q∣∣P)D_{KL}(P||Q) \neq D_{KL}(Q||P)DKL​(P∣∣Q)=DKL​(Q∣∣P)。这意味着从 PPP 到 QQQ 的“距离”和从 QQQ 到 PPP 的“距离”是不同的。因此，KL 散度不是一个真正的数学意义上的“距离”（不满足对称性和三角不等式），但它常被用作一种“伪距离”。

  2. 非负性：根据吉布斯不等式（Gibbs’ inequality），DKL(P∣∣Q)≥0D_{KL}(P||Q) \ge 0DKL​(P∣∣Q)≥0，且当且仅当 P=QP = QP=Q（几乎处处相等）时，DKL(P∣∣Q)=0D_{KL}(P||Q) = 0DKL​(P∣∣Q)=0。

应用1

  • 机器学习-分类
    训练分类模型时，通常希望模型的输出分布 QQQ 尽可能接近真实标签的分布 PPP，最小化 DKL(P∣∣Q)D_{KL}(P||Q)DKL​(P∣∣Q) 等价于最小化交叉熵 H(P,Q)H(P,Q)H(P,Q)，因为 H(P)H(P)H(P) 对于给定的数据集是常数，这就是为什么交叉熵损失函数如此常见。

应用2：决策树——信息增益

决策树用来做预测，实际场景中，能对不确定的事物进行预测，这就是价值。比如，保险公司预测客户是否续保。
决策树的目标是选择最好的特征，把数据切分得越来越纯。信息增益就是用来评价一个特征“好不好用”的指标。

  • 如何定量判断两种决策树谁优？

设特征 AAA 将数据集 DDD 划分成了 nnn 个子集 {D1,D2,...,Dn}\{D_1, D_2, ..., D_n\}{D1​,D2​,...,Dn​}，那么特征 AAA 的信息增益 IG(D,A)IG(D, A)IG(D,A) 为：

IG(D,A)=H(D)−H(D∣A)IG(D, A) = H(D) - H(D|A)IG(D,A)=H(D)−H(D∣A)

其中，H(D∣A)H(D|A)H(D∣A) 是条件熵，即按特征 AAA 划分后的加权平均熵：

H(D∣A)=∑i=1n∣Di∣∣D∣H(Di)H(D|A) = \sum_{i=1}^{n} \frac{|D_i|}{|D|} H(D_i)H(D∣A)=i=1∑n​∣D∣∣Di​∣​H(Di​)
  • ∣Di∣∣D∣\frac{|D_i|}{|D|}∣D∣∣Di​∣​：第 iii 个子集占整个数据集的权重。
  • H(Di)H(D_i)H(Di​)：第 iii 个子集内部的熵。

N-gram 语言模型

N-gram 是一种基于概率的语言模型，核心假设为：
一个词的出现只与前面 N-1 个词相关，而与其它词无关。

  • 本质是一个Markov链

因此，词序列 w1,w2,...,wmw_1, w_2, ..., w_mw1​,w2​,...,wm​ 出现的概率为：

p(w1,w2,...,wm)=p(w1)∗p(w2∣w1)....p(wm∣wm−(N−1),...,wm−1) p(w_1, w_2, ..., w_m) = p(w_1) * p(w_2|w_1) ... . p(w_m|w_{m-(N-1)}, ..., w_{m-1}) p(w1​,w2​,...,wm​)=p(w1​)∗p(w2​∣w1​)....p(wm​∣wm−(N−1)​,...,wm−1​)

实际中，N常取2(Bigram)、3(Trigram)

退化情景：当 N=1N=1N=1 时，称为一元语言模型（Unigram），此时模型简化为：

P(w1,w2,⋯ ,wm)=∏i=1mP(wi) P(w_1, w_2, \cdots, w_m) = \prod_{i=1}^m P(w_i) P(w1​,w2​,⋯,wm​)=i=1∏m​P(wi​) p(wi)p(w_i)p(wi​) 哪里来？ P(wi)=c(wi)M P(w_i) = \frac{c(w_i)}{M} P(wi​)=Mc(wi​)​
  • c(⋅)c(\cdot)c(⋅) 表示词在训练语料中出现的次数
  • MMM 是语料中词的总数

应用

  • 输入法文本补全
  • 语音识别，机器翻译
